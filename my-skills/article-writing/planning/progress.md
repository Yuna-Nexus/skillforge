## 技术决策

| 决策项 | 决定 | 理由 |
|--------|------|------|
| 配置格式 | JSON | 易于阅读和编辑 |
| 配置存储 | configs/ 目录 | 集中管理 |
| 配置加载 | 按需加载 | 减少内存占用 |
| 字段验证 | JSON Schema | 保证配置正确性 |
| reviewer_concerns.json | 迁移到 configs/ | 保留原功能 |
| /set-field | 仅初始化使用 | 不支持运行时切换 |
| 自定义配置路径 | 项目目录 / 用户目录 | 灵活配置 |
